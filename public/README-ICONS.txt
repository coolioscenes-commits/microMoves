IMPORTANT: Replace placeholder icons before production deployment

The following icon files are PLACEHOLDERS and should be replaced with proper branded icons:

Required icon files:
- icon-192.png (192x192px)
- icon-512.png (512x512px)  
- apple-touch-icon.png (180x180px minimum, recommend 512x512)
- favicon.ico (optional but recommended)

The icon.svg file is provided as a starting template. You can:
1. Edit icon.svg with your brand colors/design
2. Convert to PNG using an online tool like:
   - https://cloudconvert.com/svg-to-png
   - Or use: convert icon.svg -resize 192x192 icon-192.png
   - Or use: convert icon.svg -resize 512x512 icon-512.png

For best results, use a professional designer or icon generator service.
